# microOS
MicroOS is a minimal project and educational OS made for maximum compatibility. It can run on both old and new CPU's powered by the X86 assembly instruction set for maximum compatibility and flexibility. It is mainly made for education and i don't plan on making it a full mainstream OS. But if it gets that popular, Sure ;)

Run this to test it in a VM:
(First make sure **qemu** is installed on your testing system I highly recomend testing this on WSL or Linux. But you can also run it on windows if youre feeling a bit jolly)

```chmod +x test.sh ``` <br>
```./test.sh```
